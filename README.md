# Проект по Экологии
Проект создавался для конкурса хакатон.
## Описание
Проект это android приложение для популяризации экологии с системой наград и прогресса каждого пользователя
## Стек Технологий
<img src="https://github.com/alkmanistik/EnvironmentalProjectForHackaton-/blob/master/presentationResource/TechnologyStack.png " width="400">
IDE - Android Studio
Язык проекта - Kotlin
Язык вёрстки - Jetpack Compose
